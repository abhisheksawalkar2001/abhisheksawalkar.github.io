Data analytics is collection, transformation and organization of data in order to draw conclusions, make predictions and drive informed decision making. Data analytics is often confused with data analysis while there are related terms, they aren’t exactly the same. In fact, data analysis is a subcategory of data analytics that deals specifically with extracting meaning from data. Data analytics, as a whole, includes processes beyond analysis, including data science for using data to theorize and forecast and data engineering for building data systems
